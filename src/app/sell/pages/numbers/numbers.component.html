Decimal pipe and porcentage

Default angular pipes - Common Module

  • value after pipe:
    * {{bigNumber | number}}

  • value after pipe with argument (number: '0.2-2'):
    * {{bigNumber | number: '0.2-2'}}

  • value plus 5245254.1 after pipe with argument (number: '0.2-2'):
    * {{(bigNumber + 5245254.1) | number: '0.2-2'}}

  • value after pipe with arguments (number: '0.2-2': 'fr'):
    * {{bigNumber | number: '0.2-2': 'fr'}}

*with dollar symbol from primeNG
  • value after pipe:
    {{bigNumber | currency}}

  • value after pipe with argument (currency: 'EUR'):
    {{bigNumber | currency: 'EUR'}}

  • value after pipe with arguments (currency: 'USD': 'symbol-narrow'):
    {{bigNumber | currency: 'USD': 'symbol-narrow'}}

  • value after pipe with arguments (currency: 'EUR': 'symbol-narrow'):
    {{bigNumber | currency: 'EUR': 'symbol-narrow'}}

  • value after pipe with arguments (currency: 'EUR': 'symbol-narrow': '1.4-4'):
    {{bigNumber | currency: 'EUR': 'symbol-narrow': '1.4-4'}}
